DESCRIPTION

HTML
HTML (Hypertext Markup Language) is used to create the structure of web pages and web applications. It's the backbone of any website for the layout.
CSS
CSS (Cascading Style Sheets) is used to style and format the layout of web pages. It controls the appearance of HTML elements.
Javascript
JavaScript is a programming language used to create interactive effects within web browsers. It can be used on both the frontend and backend with technologies like Node.js.
Python
Python is a high-level programming language used for backend development, scripting, automation, data science, and artificial intelligence.
